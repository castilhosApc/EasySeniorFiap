# Assets do Mobile

Este diretório deve conter os seguintes arquivos de assets:

## Arquivos Necessários

- `icon.png` - Ícone do app (1024x1024px)
- `splash.png` - Tela de splash (1242x2436px)
- `adaptive-icon.png` - Ícone adaptativo Android (1024x1024px)
- `favicon.png` - Favicon para web (48x48px)

## Gerando Assets

Você pode usar ferramentas online ou o Expo para gerar os assets:

```bash
npx expo install expo-asset
```

Ou use o serviço do Expo:
```bash
npx expo-asset-generator
```

## Placeholder

Por enquanto, você pode usar imagens placeholder ou criar assets simples com ferramentas de design.
